# tech-study-for-interview

<br>

> 🎯 목표  
> Spring Boot 백엔드 기술 면접 대비를 위한 **모의 면접 중심 스터디**  
> 단순 개념 정리가 아닌 **면접 답변 능력 향상**을 목표로 합니다.

</br>

## 📆 스터디 운영 방식

- 비대면 진행 (주 1회)
- 인원 4명
- 주차별 Topic A / Topic B 분리
- **모든 인원 면접자 역할 필수 수행**
- 설명식 발표 ❌ / 면접 답변 방식 ⭕️

<br>

## 👥 진행 방식

1. 주차별 주제 학습 (주차별 폴더에 양식에 맞춰 스터디 및 정리)
2. 스터디 1회차에 2개의 주제가 주어짐
3. 각 2명씩 팀을 지어 1개의 주제 선택
4. 각자 배정된 주제 공부 및 상대팀 주제에 대한 질문 리스트 짜오기
5. 모의 면접 진행 시, 서로 다른 팀끼리 면접관과 지원자 역할 수행 (다대다 면접)
6. 역할 순환
   - 면접자
   - 면접관
7. 피드백
</br>


## 📝 스터디 규칙

- 면접 답변 형식으로만 말하기
- ⏱ 답변은 **2~3분 이내**
- 모르면 솔직하게 말하고 **논리적으로 추론**
- 경험이 없다면 **가상의 상황을 가정해 설명**
- 질문자는 **꼬리 질문 최소 1개 이상 필수**
